# NetDust
A clean, expressive language built for clarity — from entry point to blueprint.

## Official Libraries
 + std/: The official standard library. It is stable with no major or breaking changes.
 + ntd/: The official extension library. It is unstable and undergoes frequent updates.

## Official Net Dust Engines
| Engine | Platform | Based |
| :--- | :---: | :--- |
| **Net Dust Runtime (ndr)** | macOS / Cross platform | C++/Objective-C |
| **Noctura** | Windows Only | C# .NET |
| **Noc** | Linux Only | C++ |
| **TIT4N** | Cross platform | Noctura |
| **shimmer** | Windows Only | Noctura |
| **Endear** | Windows Only | shimmer |

> [!WARNING]
> The TIT4N(netdust 2.3), Noc, shimmer, Endear is no longer being updated.
