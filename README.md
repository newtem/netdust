# NetDust
A clean, expressive language built for clarity — from entry point to blueprint.

## Official Libraries
 + std/: The official standard library. It is stable with no major or breaking changes.
 + ntd/: The official extension library. It is unstable and undergoes frequent updates.

## Official Net Dust Engines
| Engine | Platform | Description |
| :--- | :---: | :--- |
| **Net Dust Runtime (ndr)** | macOS / Cross platform | The most standard core engine. |
| **Noctura** | Windows Only | A feature rich, all in one engine. |
| **TIT4N** | Cross platform | A simple and lightweight engine. |

> [!WARNING]
> The TIT4N engine(netdust 2.3) is no longer being updated.
