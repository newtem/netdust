# Getting Started with Netdust

Welcome to the netdust programming language! This guide will help you set up your first project and understand basic file structure.

---

## Project Initialization

To start a new project in netdust, you can use the **Net Dust Runtime (ndr)**. There are two ways to get started:

### 1. Automatic Creation (Recommended)
You can automatically scaffold a new project using the `--init` command followed by your project name:
```bash
ndr --init <name>
```

## Manual Creation

Alternatively, you can manually create a source file in your directory and start coding right away.

---

## File Extensions

netdust source files use specific extensions depending on the version.

 + Current Standard: The official file extension for netdust is .nd.
 + Legacy Extension: Previous versions up to Netdust 2.3 used the .rpp extension.

> [!WARNING]
> Support for the .rpp extension is scheduled to be discontinued in future releases.
> While .rpp files are still supported in netdust 26, it is highly recommended to migrate your files to .nd as soon as possible.
